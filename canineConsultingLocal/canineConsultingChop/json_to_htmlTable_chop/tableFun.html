<h1>Table Colors and Backgrounds</h1>

<p>
Tables can have outer border colors as well as cell border colors. They also
can have background colors, patterns, and graphic images as backdrops for the
full table or for selected rows and cells.
</p>

<h3>Table Border Colors</h3>

<p>
The appearance of border colors is different depending on the border style.
Below are example border styles displayed in red.
</p>

<blockquote>
<pre>

.outset { border:outset 5px red;}

.groove { border:groove 5px red;}
.dashed { border:dashed 5px red;}
.dotted { border:dotted 5px red;}
.solid { border:solid 5px red;}
.double {border:double 5px red;}
.inset td, .outset td, .ridge td, .groove td, .dashed td,
.dotted td, .solid td, .double td {border:inset 1px #000;}

separately with border-top, border-right, border-bottom, and border-left
colors. The following tables creates their own 3-D effects with solid borders

two tables are shown in Listing 8-17.
</p>

<blockquote>

border-left-color:#FEB9B9;}
#TAB1 td, #TAB2 td {border:inset 1px #000;}

<p>
As in the case with table borders, cells borders can be styled and colored
independently. In Figure 8.19, all cells in the two tables are styled with
solid borders 5 pixels in width. Different shades of red are applied to their
sides to create 3-D effects resembling inset and outset borders. The two
applicable style sheets are shown in Listing 8-19.
</p>

<blockquote>
<style type="text/css">
/* INSET STYLE */
table#TAB3 { border:solid 1px #000;}
table#TAB3 td { border:solid 5px red;
border-top-color:#B22222;
border-right-color:#FEB9B9;
border-bottom-color:#FEB9B9;

<h3>Deprecated Table Attributes</h3>

<p>
Border colors can be specified with the <strong>bordercolor</strong>="<i>color</i>"
attribute of the <code>&lt;table&gt;</code> tag, where <i>color</i> is a color name
or hexadecimal value.
</p>

<div class="code">&lt;table border="5" bordercolor="red"&gt;</div>
<table border="5" bordercolor="red" style="margin-top:7px">

<td>Cell 1.2</td>
<td>Cell 1.3</td>
</tr>
<tr>
<td>Cell 2.1</td>
<td>Cell 2.2</td>
<td>Cell 2.3</td>
</tr>
<tr>
<td>Cell 3.1</td>
<td>Cell 3.2</td>
<td>Cell 3.3</td>
</tr>
</table>

<p>
A colored shadow effect can be applied to a table's outer borders with the
<strong>bordercolordark</strong>="<i>color</i>" and
<strong>bordercolorlight</strong>="<i>color</i>" attributes.
</p>

<div class="code">&lt;table border="5" bordercolordark="red" bordercolorlight="pink"&gt;</div>
<table border="5" bordercolordark="red" bordercolorlight="pink" style="margin-top:7px">

</tr>
<tr>
<td>Cell 3.1</td>
<td>Cell 3.2</td>
<td>Cell 3.3</td>
</tr>
</table>

<p>
The background color of the table is given by the <strong>bgcolor</strong>="<i>color</i>"
attribute. When applied to the <code>&lt;table&gt;</code> tag, the color fills the background.
</p>

<div class="code">&lt;table border="5" bgcolor="lightblue"&gt;</div>
<table border="5" bgcolor="lightblue" style="margin-top:7px">

<tr>
<td>Cell 3.1</td>
<td>Cell 3.2</td>
<td>Cell 3.3</td>
</tr>

<p>
Cell background colors are set by applying the bgcolor attribute to a
<code>&lt;tr&gt;</code> tag (to color the row) or to a <code>&lt;td&gt;</code> tag

table background colors. The following table specifies a row color with one of
its cells taking on a different background color.
</p>

<div class="code">
